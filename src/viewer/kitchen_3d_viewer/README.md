# Kitchen 3D Viewer Prototype

This is a lightweight browser-based 3D review prototype for the high-end commercial kitchen plan.

Open `index.html` in a browser. It uses Three.js from a CDN, so internet access is required the first time the page loads. No build step is required.

## Scene Contents

- 35 ft x 45 ft room shell
- 12 in x 12 in matte gray ceramic floor tile grid
- White/light gray wall panels
- Brushed stainless steel perimeter counters
- Central cooking line
- Two 6-burner ranges
- Flat-top griddle
- Stacked combi oven
- Heavy-duty hood-type dishwasher
- Multi-tier prep islands
- Cold/dry storage shelving
- 15 ft x 4 ft stainless exhaust hood
- Ceiling lights and HVAC grilles

## Purpose

This viewer is a coordination and communication prototype. It is not yet a BIM-authoring model. The next step is to feed it from the same layout metadata generated by the AutoCAD MCP kitchen planner, then export or mirror the layout to IFC/glTF.
